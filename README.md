# Blokka's Sun and Moon Shaders

A custom Roblox shader system featuring:

- Custom Sun and Moon textures
- Sun, Moon, Blood Moon, and Combined presets
- Lighting color customization
- Day/Night slider
- Enhanced atmosphere, bloom, fog, and ambient lighting
- Original Roblox Sun and Moon hidden and replaced by custom celestial objects
- Soul Eaters Sun and Moon
## Installation

Place the script in:

```
StarterPlayer
└── StarterPlayerScripts
```

Or load remotely:

```lua
loadstring(game:HttpGet("YOUR_RAW_GITHUB_LINK"))()
```

## Features

- Sun Preset
- Moon Preset
- Blood Moon Preset
- Sun + Moon Preset
- Custom Lighting Colors
- Dynamic Sky Tracking
- Adjustable Day/Night Cycle
